API Documentation

This server provides the following endpoints:

Information Endpoints
  • /api/today - JSON data for today
  • /api/info/YYYY-MM-DD - JSON data for specific date
Image Endpoints
  • /api/artwork/today - Today's artwork
  • /api/artwork/YYYY-MM-DD - Specific date artwork
  • /api/next-artwork/today - Next available artwork (when today has none)
  • /api/next-artwork/YYYY-MM-DD - Next available artwork (when date has none)
  • /api/image/today/png - Today's generated image in PNG format
  • /api/image/today/bmp - Today's generated image in BMP format
  • /api/image/YYYY-MM-DD/png - Specific date generated image in PNG format
  • /api/image/YYYY-MM-DD/bmp - Specific date generated image in BMP format
Web Pages
  • /today - Today's liturgical information page
  • /date/YYYY-MM-DD - Specific date liturgical information page
